获取 Git 仓库

有两种取得 Git 项目仓库的方法。

  • 第一种是在现有项目或目录下导入所有文件到 Git 中
  • 第二种是从一个服务器克隆一个现有的 Git 仓库

克隆现有的仓库

如果你想获得一份已经存在了的Git仓库的拷贝，比如说，你想为某个开源项目贡献自己的一份力，这时就要用到 git clone 命令。

克隆仓库的命令格式是 git clone [url] 。
