Sentinel 主流框架的适配

适配

为了减少开发的复杂程度，Sentinel对大部分的主流框架，例如 Web Servlet、Dubbo、Spring Cloud、gRPC、Spring WebFlux、Reactor 等都做了适配。您只需要引入对应的依赖即可方便地整合 Sentinel。
包括以下配置：

Web 适配

  • Spring Boot/Spring Cloud
  • Web Servlet
  • Spring WebFlux

    RPC 适配

  • Apache Dubbo
  • gRPC
  • Feign
  • SOFARPC

    HTTP client 适配

  • Apache HttpClient
  • OkHttp
  • Reactive 适配
  • Reactor

    API Gateway 适配

  • Spring Cloud Gateway
  • Netflix Zuul 1.x
  • Netflix Zuul 2.x
  • Apache RocketMQ
